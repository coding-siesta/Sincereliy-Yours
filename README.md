# Sincereliy-Yours
A flower shop mobile application created using Flutter, Dart, and Firebase

## About
Sincereliy Yours is a transactional mobile app that allows user to browse and order flower arrangement online. Built as a school project using Flutter, Dart, and Firebase

## Features
- Browse flower products
- Add items to cart
- Place and manage orders
- Firebase for saving user registration
